# [55. 跳跃游戏](https://leetcode.cn/problems/jump-game/description/)

```cpp
class Solution {
public:
    bool canJump(vector<int>& nums) {
        int k = 0;
        for (int i = 0; i < nums.size(); i++) {
            if (i > k) return false;
            k = max(k, i + nums[i]);
        }
        return true;
    }
};
```
